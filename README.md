# Online Shop – JavaScript Frameworks Assignment

This project is an online shop built with **Next.js, React, TypeScript, and Tailwind CSS**.  
The application fetches products from the Noroff Online Shop API and allows users to browse products, view product details, manage a shopping cart, and complete a checkout flow.

---

## Live Site
https://mikkel-jsf-shop.netlify.app

## Repository
https://github.com/NoroffFEU/jsfw-2025-v1-mikkel-a-a-jsframeworks

---

## Features

- Product listing fetched from API
- Product details page
- Search functionality
- Product sorting (price and rating)
- Shopping cart with quantity management
- Checkout success flow
- Contact form with validation
- Responsive design for mobile and desktop

---

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS
- REST API
